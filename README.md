# Web Corporativa de FoodLogístic S.A.

Aquesta és la web corporativa de **FoodLogístic S.A.**, una empresa fictícia dedicada al sector alimentari que necessitava modernitzar la seva infraestructura IT i presència digital, realitzada dins el mòdul *Projecte Intermodular*.

## Demostració

[🌐 URL de la web](https://vramills.github.io/web-corporativa/)

## Tecnologies Utilitzades

Per la creació d'aquesta web, s'han utilitzat les següents tecnologies i eines:

- **Eina IA:** Gemini 3 Flash (per a la generació de l'estructura inicial i el prototipat de codi).
- **HTML5/CSS3:** Utilitzant etiquetes semàntiques i disseny responsive (Flexbox/Grid) per a una visualització minimalista i professional.
- **Vanilla JavaScript:** Per a la gestió funcional del banner de cookies i l'experiència d'usuari.
- **Statcounter:** Utilitzat per a l'analítica web i el seguiment de visites.

## Descripció del Projecte

El projecte resol els problemes de FoodLogístic S.A. mitjançant:
1. **Infraestructura:** Proposta de sistemes redundants d'alta disponibilitat.
2. **Seguretat:** Implementació de polítiques LOPD/RGPD.
3. **Web:** Landing page moderna que compleix la normativa LSSI.
